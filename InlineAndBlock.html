paragraph is a block element it means if we write something inside p tag and then again write something in another p tag then that will start from next line because, The Block element means that it will take the full width of a single line and does not allow any other content to fit in it

Whereas, span tag is a inline element it means if we write in two different span tag then it will come in same line .
